03 / Core Concept

Paging — In Depth

Fixed-size memory blocks for efficient allocation

Paging divides a process into pages and physical memory into equal-sized frames. Pages are mapped to frames anywhere in RAM using a Page Table. Page Size = Frame Size.

04 / Virtual Memory & Summary

Virtual Memory & Conclusion

💾 Virtual Memory

Virtual Memory allows programs larger than RAM to execute by using disk storage as an extension of RAM. Paging is the key mechanism that makes virtual memory possible.

05 / Diagram

Address Translation in Paging

How a logical address is translated to a physical address

🔢
Logical Address

Generated by the CPU. Split into two parts: Page Number (p) and Page Offset (d).

📋
Page Table Lookup

Page number p indexes into the Page Table to fetch the Frame Number (f).

📍
Physical Address

Physical address = Frame Number × Frame Size + Offset (d). This points to the exact memory location.
